設問3<br>

input filenameと出力されるので、監視ログファイルの名前を入力する<br>
input Nと出力されるので、何回以上連続してタイムアウトした場合のみ故障と満たしたいかを入力<br>
input m tと出力されるので、直近m回の平均応答時間がtミリ秒を超えた時に過負荷状態になるようにみなしたいかをそれぞれ入力<br>
